# MCLA v0.1.0

## 🎉 首个正式版本发布

MCLA (Minecraft Launcher Assistant) 是一款基于 Electron + Vue 3 开发的现代化 Minecraft 启动器。

---

## ✨ 核心功能

### 账户系统
- ✅ Microsoft OAuth 设备码登录流程
- ✅ 离线模式支持
- ✅ 多账户切换管理
- ✅ Token 自动刷新

### 游戏版本
- ✅ 官方版本下载与安装
- ✅ Forge / Fabric / Quilt 模组加载器自动安装
- ✅ 版本隔离管理
- ✅ 自定义 JVM 参数

### Mod 下载
- ✅ CurseForge 资源搜索与下载
- ✅ Modrinth 资源搜索与下载
- ✅ 自动解析依赖关系
- ✅ 版本兼容性检查

### 游戏启动
- ✅ 一键启动游戏
- ✅ 自动 Java 检测与下载
- ✅ 内存自动配置
- ✅ 游戏日志实时查看

---

## 📦 安装包说明

| 文件名 | 大小 | 说明 |
|--------|------|------|
| `MCLA Setup 0.1.0.exe` | ~92 MB | Windows 安装程序（推荐） |
| `MCLA-0.1.0-win-portable.zip` | ~134 MB | 便携版（无需安装，解压即用） |

---

## 🚀 快速开始

### 方式一：安装版
1. 下载 `MCLA Setup 0.1.0.exe`
2. 双击运行安装程序
3. 按向导完成安装
4. 从开始菜单或桌面快捷方式启动

### 方式二：便携版
1. 下载 `MCLA-0.1.0-win-portable.zip`
2. 解压到任意目录
3. 运行 `MCLA.exe`

---

## 📝 系统要求

- **操作系统**: Windows 10/11 (64位)
- **内存**: 4GB 及以上
- **存储**: 至少 500MB 可用空间（游戏本体需额外空间）
- **网络**: 需要网络连接用于登录和下载资源

---

## ⚠️ 注意事项

1. **首次启动**需要登录 Microsoft 账户（正版验证）
2. **游戏目录**默认位于 `%APPDATA%/MCLA`，可在设置中修改
3. **Java 自动下载**功能需要网络连接
4. **Mod 下载**功能依赖 CurseForge 和 Modrinth API
